## CONSOLIDATED DESIGN SYSTEM FOR MANUS ECOSYSTEM

### Final Unified Palette

| Role | Color | Hex Code | Usage |
|------|-------|----------|-------|
| Primary | Teal | #009688 | Lingueefy brand, links, icons |
| CTA Primary | Orange | #F5A623 | All primary action buttons |
| CTA Hover | Dark Orange | #E09620 | Button hover states |
| Background Dark | Dark Teal | #1a3a3a | Hero sections, headers |
| Background Light | White | #FFFFFF | Cards, content areas |
| Background Accent | Sage | #8fa68f | Subtle backgrounds |
| Text Primary | Dark | #1a1a1a | Body text on light |
| Text Light | White | #FFFFFF | Text on dark backgrounds |
| Text Muted | Gray | #6b7280 | Secondary text |
| Barholex Accent | Gold | #D4A853 | Barholex Media brand |
| Success | Green | #22c55e | Success states |
| Error | Red | #ef4444 | Error states |

### Typography System

| Element | Font | Weight | Size |
|---------|------|--------|------|
| H1 | System/Inter | Bold (700) | 48-64px |
| H2 | System/Inter | Bold (700) | 36-48px |
| H3 | System/Inter | Semibold (600) | 24-32px |
| Body | System/Inter | Regular (400) | 16-18px |
| Small | System/Inter | Regular (400) | 14px |
| CTA | System/Inter | Bold (700) | 16-18px |

### Component Rules

1. **Primary CTA Buttons**: Orange background (#F5A623), white text, rounded-lg, shadow-md
2. **Secondary Buttons**: Teal outline or white background with teal text
3. **Cards**: White background, rounded-xl, shadow-lg, subtle border
4. **Hero Sections**: Dark teal gradient background, white text, orange CTA
5. **Navigation**: Clean, minimal, with brand toggle visible
6. **Testimonials**: Photo + name + title + LinkedIn link
7. **Pricing Cards**: Clear hierarchy, orange "Book Now" CTA
