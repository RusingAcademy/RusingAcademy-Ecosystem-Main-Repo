### The 6 Structured Learning Paths:

#### PATH I: Foundations (A1 → A1+)
- **Target**: Complete beginners
- **Goal**: Build foundational workplace French
- **Modules**:
  - M1: Premiers Pas Professionnels (First Professional Steps)
  - M2: Communication Quotidienne (Daily Communication)
  - M3: Interactions Essentielles (Essential Interactions)
  - M4: Vers l'Autonomie (Toward Autonomy)

#### PATH II: Elementary (A2)
- **Target**: Basic learners
- **Goal**: Develop elementary workplace communication
- **Focus**: Expanding vocabulary, basic conversations, simple emails

#### PATH III: Intermediate (B1) → BBB Target
- **Target**: Intermediate learners
- **Goal**: Achieve BBB level for federal positions
- **Focus**: Professional discussions, meeting participation, written reports

#### PATH IV: Upper Intermediate (B2) → CBC Target
- **Target**: Upper intermediate learners
- **Goal**: Achieve CBC level for bilingual positions
- **Focus**: Complex discussions, presentations, nuanced communication

#### PATH V: Advanced (C1) → CCC Target
- **Target**: Advanced learners
- **Goal**: Achieve CCC level for executive positions
- **Focus**: Leadership communication, strategic discussions, policy briefings

#### PATH VI: Mastery (C1+/C2)
- **Target**: Near-native proficiency
- **Goal**: Full bilingual mastery
- **Focus**: Nuanced expression, cultural fluency, executive presence

### Key Features of Each Path:
- 4 modules per path
- 4 lessons per module
- Progressive skill building
- GC workplace context
- Oral and written components
- Self-assessment checkpoints
